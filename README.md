# Global-YouTube-Statistics-Dashboard
Power BI dashboard analyzing global YouTube statistics.

# Project Overview
This project analyzes global YouTube channel data to understand trends, performance, and growth patterns across different categories and countries. The dashboard provides insights into subscriber distribution, channel types, uploads, and overall platform growth.
The analysis was performed using Microsoft Power BI.

# Problem Statement
YouTube is one of the largest digital content platforms in the world. With thousands of creators across multiple categories and countries, it becomes important to analyze which factors contribute to channel success. This project aims to identify top-performing channels, dominant categories, and key metrics influencing growth and popularity.

# Objectives
- Identify the top YouTube channels by subscribers.
- Analyze which category has the highest total subscribers.
- Determine which country has the most successful channels.
- Examine the relationship between number of uploads and subscribers.
- Analyze the growth of YouTube channels over the years.
- Calculate the average subscriber count across channels.

# Key Analytical Questions
1. Which are the top 10 YouTube channels by subscribers?
2. Which category has the highest total subscribers?
3. Which country has the most successful channels?
4. Is there a relationship between uploads and subscriber count?
5. How has the number of YouTube channels grown over the years?
6. What is the average subscriber count across channels?
   
# Tools Used
- Microsoft Power BI
- Data Cleaning & Transformation
- Data Visualization Techniques
  
# Conclusion
The dashboard provides insights into YouTube channel performance, highlighting key factors such as category dominance, country contribution, and content volume impact on subscribers. The analysis helps understand patterns behind successful YouTube channels.
